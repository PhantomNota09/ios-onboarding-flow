# iOS Onboarding Flow

A clean, modern iOS onboarding experience built with UIKit and Storyboards, featuring user profile setup and push notification permissions.

## Overview

This project demonstrates a complete onboarding flow for iOS applications, guiding new users through account setup, photo upload, and notification permissions. The app showcases fundamental iOS development patterns and user experience best practices.

## Features

### Onboarding Screens
- **Welcome Screen**: Initial landing page with app title
- **Name & Username Input**: Text field-based user information collection
- **Photo Upload**: Image picker integration with visual preview
- **Push Notifications**: Permission request with clear value proposition

### UI Components
- Navigation Controller for flow management
- Custom text fields with placeholder styling
- Image picker and photo display
- Rounded action buttons with consistent styling
- Icon graphics for visual communication
- Multi-screen navigation flow

### Technical Implementation
- **Navigation**: Navigation controller managing screen transitions
- **Text Input**: UITextField with custom styling and validation-ready structure
- **Layout**: Auto Layout constraints for responsive design
- **Buttons**: Custom styled UIButtons with rounded corners and tap actions
